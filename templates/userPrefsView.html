User preferences

Icon

The image icon is from your gravatar, or globally recognizable avatar. If you haven't created one of these on gravatar.com you will have a semi-random pattern derived from your email address. This icon is used to represent you in various places on this textus server, if you wish to change it please use gravatar's web interface.
 

Annotation Colour

This control determines the colour used for any annotations you make. Changes here will be reflected in any annotations you have already made, assuming the viewer has selected an annotation colouring mode based on author.

 

Display Name

Used in place of an email address, if you leave this field blank textus will use your email address when it needs to represent you in some fashion.
